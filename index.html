JavaScript

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

operators

The definition of an operator is a symbol that is used to perform an operation.

variable

a data item that may take on more than one value during the runtime of a program.

function

A function is a block of organized, reusable code that is used to perform a single, related action.

method

A function associated with an object.

argument

a value or address passed to a procedure or function at the time of call.
